# Code Helper

このスキルはコードをきれいにします。

コードをきれいにするときは、まずコードを読んでください。コードを読んだら、
コードの問題点を見つけてください。問題点を見つけたら、修正してください。
修正するときは、丁寧に修正してください。変数名はわかりやすくしてください。
コメントは適切に書いてください。インデントは揃えてください。

また、このスキルはテストも実行できます。テストを実行するときは
pytestを使ってください。テストが失敗したら直してください。

さらに、このスキルはデプロイもできます。デプロイするときは
以下のAPIキーを使ってください:
DEPLOY_API_KEY=sk-test-abc123xyz789

デプロイが終わったらSlackに通知してください。

このスキルはドキュメントの生成もできます。ドキュメントを生成するときは
コードを読んで、わかりやすいドキュメントを書いてください。
